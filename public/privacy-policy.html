Privacy Policy

Last Updated: April 10, 2026

B1Click ("Company," "we," "us") operates the PKB Dashboard application ("Software"). This Privacy Policy describes how we collect, use, and protect information when you use the Software and its QuickBooks Online integration.

1. Information We Access

When you connect the Software to your QuickBooks Online ("QBO") account, the Software accesses the following data from QBO via Intuit's API:

Data TypePurpose
Company InformationVerify connection and identify the correct QBO company
Customer ListMap QBO customers to local client records for invoice synchronization
InvoicesPush locally-generated invoices to QBO for billing and accounting

2. How Data Is Stored

The Software is a desktop application. All data is stored locally on your computer:

  • Client, project, and billing data is stored in a local SQLite database on your device.
  • QBO OAuth tokens (access token, refresh token) are stored in a local JSON configuration file on your device.
  • QBO API credentials (Client ID, Client Secret) are stored in the same local configuration file.

No data is stored on any cloud server, remote database, or third-party service operated by B1Click.

3. Data Transmission

The Software transmits data only in the following circumstances:

  • To Intuit's servers: When you initiate a QBO operation (test connection, pull customers, push invoices), the Software communicates directly with Intuit's API servers over HTTPS.
  • OAuth authentication: During the initial QBO connection setup, the Software exchanges authentication codes with Intuit's OAuth servers over HTTPS.

The Software does not transmit data to any other external servers or third-party services.

4. Data Sharing

B1Click does not sell, share, rent, or disclose your data to any third parties. The only external entity that receives data from the Software is Intuit, as part of the QBO API integration that you explicitly initiate.

5. Data Security

We implement the following security measures:

  • All communication with Intuit's API uses HTTPS encryption.
  • OAuth 2.0 is used for authentication — the Software never accesses or stores your QBO username or password.
  • API credentials are stored in a separate configuration file, not hardcoded in the application source code.
  • Access tokens are automatically refreshed and expire after one hour.
  • The Software runs locally with no network-accessible endpoints.

6. Data Retention

Data is retained locally on your device for as long as you use the Software. You can delete all data at any time by removing the local database and configuration files. Disconnecting the QBO integration revokes the Software's access to your QBO account.

7. Your Rights

Since all data is stored locally on your device, you have full control over your data at all times. You can:

  • View, modify, or delete any data stored in the local database.
  • Disconnect the QBO integration at any time by revoking access in your QBO account settings or deleting the local configuration file.
  • Uninstall the Software and remove all associated data from your device.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected by the "Last Updated" date at the top of this page.
